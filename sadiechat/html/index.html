Conversational User Interface (CUI)

Messaging apps are everywhere. Two of the greatest benefits are convenience and adaptability. With nothing to download, no need to login, and no user interface to learn, users can tell me what they want in their own words.

What kind of things can we talk about?

Never forget to take your medications again. Ask me to create a medication reminder. Worried about side effects? I can research and share information with you. Medications Information, Reminders, and Side Effects
Not feeling well? I can help to identify possible causes and treatments, as well as encourage you to talk to your healthcare provider. Common Symptoms Possible Causes and Treatments
I can help you better understand your lab tests and results. Ask me about them. Lab Tests and Results Information
Need to see your provider? No problem, I can help with scheduling an appointment, creating an appointment reminder, messaging the practice, and speeding up check-in when you arrive. Appointments Schedule, Reminders, Check-In, and Messaging

Add Sadie to Microsoft Teams

1. You will need Sadie's Microsoft app ID. (copy & paste)

66368b4b-8c37-492a-95c3-dd0509d5d433

2. From within Microsoft Teams, on the Chat pane, select the Add chat icon. For To:, paste Sadie's' Microsoft app ID

3. The app ID should resolve to Sadie. Select Sadie and send a message to initiate a conversation

Alternatively, you can paste Sadie's app ID in the search box in the top left in Microsoft Teams.

In the search results page, navigate to the People tab to see Sadie and to start chatting with her.

Add Sadie to Skype

Note: Skype for Business does not support the Microsoft Bot Framework, but you can use consumer Skype.

1. You will need Sadie's link. (copy & paste)

https://join.skype.com/bot/66368b4b-8c37-492a-95c3-dd0509d5d433

2. Paste Sadie's link in a web browser.

3. The browser should prompt you to add Sadie to Contacts. Select the 'Add to Contacts' button to download or login to Skype.

4. Select Sadie from Contacts and start a conversation.
